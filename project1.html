How to Use This Application


This application visualizes data from the website https://aqs.epa.gov/aqsweb/airdata/download_files.html. You can analyze the air quality data around the US from 1980 – 2018. To select different options such as the year, pollutant type, state and county, select from the list options on the bar on the right-hand side of the page. As you select new options the data on the screen should update to visualize the specifications you chose.

To visualize the AQI days in your chosen county and state look at the pie chart displayed in the right-hand corner. The tab options above this chart allows you to see the same data visualized as a bar graph and a table. Select any of the tab options to see different display types.

To visualize the pollutant days percentages, look at the pie chart on the left-hand size of the display. To visualize different pollutants, select a different pollutant from the options bar on the left-hand side of the page. The tab options above this chart allows you to see the same data visualized as a bar graph and a table. Select any of the tab options to see different display types.

Below the pie charts are the line graph charts. You can see how the Max AQI, Median AQI and 90th Percentile AQI data changes from 1980 – 2018 for the county and state you chose. Select the tab option to see the pollutant line graph. This graph displays the percentages of days for pollutants from 1980 – 2018 for the county and state you chose.

Below the line graphs you can see an interactive map that displays the county location you selected as a red dot. The map will adjust its zoom and location to the correct location on the map for the county you selected.



Data


The data visualized in this project can be found at https://aqs.epa.gov/aqsweb/airdata/download_files.html. I downloaded all the AQI by County files. Additionally, I downloaded the Site Listing files to get the latitude and longitude locations of each county. I used the csv files to create the visualizations you see on the application.

Getting Started


  • Download & Install R from https://www.r-project.org/
  • Download & Install RStudio from https://www.rstudio.com/products/rstudio/download/
  • Open RStudio
  • Install the following packages by entering the following in the console:

    install.packages("shiny")
    install.packages("shinydashboard")
    install.packages("ggplot2")
    install.packages("lubridate")
    install.packages("DT")
    install.packages("leaflet")
    install.packages("shinyalert")
    install.packages("dplyr")
    install.packages("tidyverse")

  • Unzip files and open application
  • To run the application press the “Run Application” button on the top right hand side
